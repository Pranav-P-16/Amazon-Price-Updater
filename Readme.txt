# How to Use
--------------
* Simply open the file 'Link.txt' and paste your link there and Save and Exit (Only 1 link allowed at a time)

* Now run 'Amazon Price Updater' :)

* Also, Displayed prices will be stored to 'Saved_Prices.txt' located in the folder 'Price_Data' ;)
